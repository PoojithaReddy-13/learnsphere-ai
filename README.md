# LearnSphere AI 🎓

AI-powered personalized learning assistant built using Next.js, TypeScript, Tailwind CSS, and Google Gemini AI.

## Problem Statement

Students often struggle to navigate lengthy notes, PDFs, and study materials while preparing for exams. LearnSphere AI transforms traditional learning resources into an interactive AI-powered learning experience.

## Features

* AI Chat Assistant
* PDF Upload Support
* Quiz Generation
* Content Summarization
* Personalized Learning Assistance
* Responsive User Interface

## Tech Stack

### Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS

### Backend

* Next.js API Routes

### AI

* Google Gemini API

## Project Workflow

1. Upload learning material.
2. Ask questions using AI Chat.
3. Generate quizzes for self-assessment.
4. Use summarization for quick revision.
5. Improve learning through AI-powered assistance.

## Installation

```bash
npm install
npm run dev
```
